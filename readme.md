## OPTICAL CHARACTER RECOGNITION ##

This project uses `streamlit` for web app development and `easyocr` for Optical Character Recognition (to extract text from images).

......................................................
Required Libraries:
* `streamlit` 
* `easyocr` 
* `numpy` 
* `Pillow` - https://python-pillow.org/
